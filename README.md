# 🏡 Global Airbnb Performance Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard that analyzes Airbnb listings across major global cities. The dashboard provides valuable insights into market share, pricing, ratings, property types, host verification, and listing trends.

## 📊 Dashboard Features
- Market Share by City
- Listing Growth Over Time
- Average Ratings by City
- Property Type Distribution
- Average Price by Room Type
- Review Trends
- Host Verification Analysis

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Key Insights
- Paris, New York, and Sydney contribute the highest number of listings.
- Entire Home/Apartment is the most popular property type.
- Mexico City has the highest average customer rating.
- Listing growth peaked around 2015–2016.
- Most Airbnb hosts are identity verified.
